# extraneous

> Внешние источники данных НКБ

## Окружение

* node.js
* npm
* grunt-cli `npm install grunt-cli -g`

## Инициализация

Установка зависимостей проекта

    npm install


## Сборка

Проверка кода, тесты

    grunt build


## Проверка кода

    grunt jshint


## Запуск тестов

    grunt test


## Очистка

Удаление зависимостей проекта

    grunt clean
